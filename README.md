# HR Dashboard

A modern HR Performance Management Dashboard built with Next.js, React, and Tailwind CSS. Track employee performance, manage bookmarks, and view analytical insights with a beautiful, responsive interface.

##  Features

### Core Features
- ** Employee Directory**
  - Browse employee profiles with performance ratings
  - Real-time data fetching from DummyJSON API
  - Performance badges and star ratings

- ** Advanced Search & Filter**
  - Search by name, email, or department
  - Multi-select filters for departments and ratings
  - Instant results with debounced search

- ** Detailed Employee Profiles**
  - Comprehensive employee information
  - Tabbed interface (Overview, Projects, Feedback)
  - Performance history tracking

- ** Bookmark System**
  - Save favorite employees for quick access
  - Persistent storage using localStorage
  - Bulk actions on bookmarked employees

- ** Analytics Dashboard**
  - Department-wise performance charts
  - Bookmark trends visualization
  - Performance distribution insights
  - Top performers leaderboard

- ** Dark/Light Mode**
  - System-wide theme toggle
  - Persistent theme preference
  - Smooth transitions

### Additional Features
 ** Create Employee Modal** - Add new employees with form validation
 ** Fully Responsive** - Works seamlessly on mobile, tablet, and desktop
 ** Performance Optimized** - Fast loading with Next.js optimization
 ** Modern UI/UX** - Clean design with Tailwind CSS
 ** Modular Architecture** - Reusable components and custom hooks

##   Tech Stack

 **Framework:** [Next.js 15.4](https://nextjs.org/) (App Router)
 **UI Library:** [React 19](https://react.dev/)
 **Styling:** [Tailwind CSS 3.4](https://tailwindcss.com/)
 **State Management:** React Context API
 **Charts:** [Chart.js](https://www.chartjs.org/) with react-chartjs-2
 **Animations:** [Framer Motion](https://www.framer.com/motion/)
 **Data Source:** [DummyJSON API](https://dummyjson.com/)
 **Deployment:** [Vercel](https://vercel.com/)

##  Getting Started

### Prerequisites

- Node.js 18.17 or later
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/ankitkommalapati/hr-dashboard.git
   cd hr-dashboard/hr-dashboard

2. **Install Dependencies**
   ```bash
   npm install
   # or
   yarn install

3. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev

4. **Open your browser**
   Navigate to http://localhost:3000

## Usage

### Search Employees
* Use the search bar to find employees by name, email, or department.
* Results update in real-time as you type.

### Bookmark Employees
* Click the Filters button to access department and rating filters.
* Select multiple options for refined results.
* Clear all filters with one click.

### Bookmark Employees
* Click "Bookmark" on any employee card.
* Access all bookmarks from the Bookmarks page.
* Perform bulk actions like "Promote" or "Assign to Project".

### View Analytics
* Navigate to Analytics to see performance insights.
* Interactive charts show department performance and trends.
* View top performers across the organization.

### Toggle Theme
* Click the theme toggle in the header.
* Preference is saved and persists across sessions.
